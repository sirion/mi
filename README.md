# mi library

Assorted code that fit nowhere else.

## File structure

```text
+
+-+ js          - JavaScript 
| + js/modules  - Javascript es6 modules
|
+-+ rs          - Rust
| + rs/binary   - crate for handling binary data
| + rs/http     - crate providing an http server
| + rs/image    - helper crate for resizing images

```
